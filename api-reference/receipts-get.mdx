---
title: GET /v1/receipts/{id}
description: Retrieve a full decision receipt with complete audit data
---

Retrieve a single decision receipt with complete audit data — evidence, exclusions, conflicts, channel contributions, and token accounting.

## Request

```bash
GET /v1/receipts/{receipt_id}
```

<ParamField path="receipt_id" type="string" required>
  UUID of the receipt.
</ParamField>

## Response

<ResponseField name="id" type="string">Receipt UUID.</ResponseField>
<ResponseField name="query" type="string">The original query.</ResponseField>
<ResponseField name="timestamp" type="string">ISO 8601 creation time.</ResponseField>
<ResponseField name="workspace_id" type="string">Workspace scope.</ResponseField>
<ResponseField name="actor_id" type="string">Actor who made the request.</ResponseField>

<ResponseField name="evidence" type="Citation[]">
  The citations that made it into the context pack.
</ResponseField>

<ResponseField name="channel_contributions" type="object[]">
  How many candidates each retrieval channel produced and how many were selected.
  <Expandable title="properties">
    <ResponseField name="channel" type="string">Channel name: `lexical`, `semantic`, or `temporal`.</ResponseField>
    <ResponseField name="candidate_count" type="integer">Candidates produced by this channel.</ResponseField>
    <ResponseField name="selected_count" type="integer">Candidates that made it into the final pack.</ResponseField>
    <ResponseField name="weight" type="float">Channel weight used for fusion.</ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="exclusions" type="object[]">
  Every span that was considered but excluded.
  <Expandable title="properties">
    <ResponseField name="span_id" type="string">UUID of the excluded span.</ResponseField>
    <ResponseField name="artifact_id" type="string">UUID of the source artifact.</ResponseField>
    <ResponseField name="reason" type="string">One of: `BUDGET`, `POLICY`, `REDUNDANCY`, `LOW_RELEVANCE`, `PRIVATE`, `MARKED_WRONG`, `OUTDATED`.</ResponseField>
    <ResponseField name="detail" type="string">Human-readable explanation.</ResponseField>
    <ResponseField name="relevance_score" type="float">The span's relevance score when it was excluded.</ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="conflicts" type="object[]">
  Detected contradictions between included evidence spans.
  <Expandable title="properties">
    <ResponseField name="id" type="string">Conflict UUID.</ResponseField>
    <ResponseField name="claim_a" type="object">First conflicting span reference.</ResponseField>
    <ResponseField name="claim_b" type="object">Second conflicting span reference.</ResponseField>
    <ResponseField name="description" type="string">Human-readable conflict description.</ResponseField>
    <ResponseField name="resolution" type="string | null">Resolution if one exists.</ResponseField>
    <ResponseField name="resolved_by" type="string | null">Actor who resolved it.</ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="token_accounting" type="object">
  Exact token budget breakdown.
  <Expandable title="properties">
    <ResponseField name="budget" type="integer">Requested token budget.</ResponseField>
    <ResponseField name="prefix_tokens" type="integer">Tokens used by pinned/authoritative content.</ResponseField>
    <ResponseField name="working_set_tokens" type="integer">Tokens used by query-specific evidence.</ResponseField>
    <ResponseField name="total_used" type="integer">Total tokens in the pack.</ResponseField>
    <ResponseField name="remaining" type="integer">Unused budget.</ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="confidence" type="float">Confidence score (0.0 - 1.0).</ResponseField>
<ResponseField name="abstain_flag" type="boolean">Whether the system abstained.</ResponseField>
<ResponseField name="abstain_reason" type="string | null">Reason for abstention.</ResponseField>
<ResponseField name="assumptions" type="string[]">Any degradation notes (e.g., "Semantic search unavailable").</ResponseField>
<ResponseField name="policy_summary" type="object">Policy that was applied.</ResponseField>
<ResponseField name="pack_hash" type="string">SHA-256 hash of the context pack output.</ResponseField>

## Example

<CodeGroup>

```python Python
receipt = client.get_receipt("r1a2b3c4-...")

# Token budget breakdown
acct = receipt["token_accounting"]
print(f"Budget: {acct['budget']}, Used: {acct['total_used']}, Remaining: {acct['remaining']}")

# What was excluded and why
for exc in receipt["exclusions"]:
    print(f"  Excluded: {exc['reason']} — {exc['detail']}")

# Detected conflicts
for conflict in receipt["conflicts"]:
    print(f"  Conflict: {conflict['description']}")
```

## Exclusion Reasons

| Reason | Description |
|---|---|
| `BUDGET` | Span was relevant but the token budget was full |
| `POLICY` | Span was blocked by the request's access policy |
| `REDUNDANCY` | Span was too similar to an already-included span |
| `LOW_RELEVANCE` | Span scored below the relevance threshold |
| `PRIVATE` | Span was marked private via feedback |
| `MARKED_WRONG` | Span was flagged as incorrect via feedback |
| `OUTDATED` | Span was flagged as outdated via feedback |
