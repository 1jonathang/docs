## Install the SDK

<CodeGroup>

```bash pip
pip install nined-memory
```

```bash npm
npm install @9d/memory
```

</CodeGroup>

## Set Up Your Client

Sign up at [9dlabs.xyz/signup](https://9dlabs.xyz/signup) to get an API key. Your base URL is `https://api.9dlabs.xyz`.

```python
from nined.memory import MemoryClient

client = MemoryClient(base_url="https://api.9dlabs.xyz", api_key="your-key")
```

## Store Your First Artifact

Ingest a document into a workspace:

<CodeGroup>

```python Python
client.ingest(
    workspace_id="my-app",
    actor_id="system",
    artifact_type="document",
    raw_payload={
        "title": "Refund Policy",
        "content": "Refunds are accepted within 30 days of purchase. No refunds after 30 days."
    },
    idempotency_key="doc_refund_policy_v1",
)
```

```typescript TypeScript
import { MemoryClient } from "@9d/memory";

const client = new MemoryClient("https://api.9dlabs.xyz", "your-key");

await client.ingest({
  workspace_id: "my-app",
  actor_id: "system",
  artifact_type: "document",
  raw_payload: {
    title: "Refund Policy",
    content: "Refunds are accepted within 30 days of purchase. No refunds after 30 days."
  },
  idempotency_key: "doc_refund_policy_v1",
});
```

```bash curl
curl -X POST https://api.9dlabs.xyz/v1/ingest \
  -H "Content-Type: application/json" \
  -d '{
    "workspace_id": "my-app",
    "actor_id": "system",
    "artifact_type": "document",
    "raw_payload": {
      "title": "Refund Policy",
      "content": "Refunds are accepted within 30 days of purchase. No refunds after 30 days."
    },
    "idempotency_key": "doc_refund_policy_v1"
  }'
```

</CodeGroup>

## Retrieve a Context Pack

Query for relevant context within a token budget:

<CodeGroup>

```python Python
pack = client.context_pack(
    query="Can I get a refund for a 45-day-old purchase?",
    workspace_id="my-app",
    max_tokens=1024,
)

if pack["abstain_flag"]:
    print("Insufficient memory to answer.")
else:
    print(pack["context_text"])       # Ready to inject into your LLM
    print(f"Confidence: {pack['confidence']:.0%}")
    print(f"Receipt: {pack['receipt_id']}")
```

```typescript TypeScript
const pack = await client.contextPack({
  query: "Can I get a refund for a 45-day-old purchase?",
  workspace_id: "my-app",
  max_tokens: 1024,
});

console.log(pack.context_text);
console.log(`Confidence: ${pack.confidence}`);
```

```bash curl
curl -X POST https://api.9dlabs.xyz/v1/context-pack \
  -H "Content-Type: application/json" \
  -d '{
    "query": "Can I get a refund for a 45-day-old purchase?",
    "max_tokens": 1024,
    "workspace_id": "my-app"
  }'
```

</CodeGroup>

## Use the Context in Your LLM

The `context_text` field is a ready-to-inject prompt string with `[cite:N]` markers:

```python
from openai import OpenAI

oai = OpenAI()

completion = oai.chat.completions.create(
    model="gpt-4o-mini",
    messages=[
        {
            "role": "system",
            "content": (
                "Answer using ONLY the provided context. "
                "Cite sources using [cite:N] format.\n\n"
                + pack["context_text"]
            ),
        },
        {"role": "user", "content": "Can I get a refund for a 45-day-old purchase?"},
    ],
    temperature=0,
)

print(completion.choices[0].message.content)
```
